# Claude Code 프롬프트 카드
### PHK Finance팀 | 복붙해서 바로 쓰세요

> **사용법**: VSCode 왼쪽 Claude Code 탭 열기 → 아래 템플릿 복사 → `[ ]` 괄호 안만 바꿔서 입력

---

## ✉️ 메일 작성

```
너는 피에이치코리아 재무팀 담당자야.
[거래처 담당자 / 내부 팀장 / 임원]에게 [메일 목적]을 전달하는 메일을 써줘.
- 톤: 정중하고 간결하게
- 분량: 5문장 이내
- 핵심 내용: [한 줄로 설명]
```

---

## 📋 메일 · 문서 요약

```
아래 내용을 읽고 핵심 3가지와 내가 해야 할 액션을 정리해줘.
숫자·날짜·담당자는 빠짐없이 포함.

[메일 또는 문서 내용 붙여넣기]
```

---

## ↩️ 답장 초안

```
아래 메일에 [수락 / 거절 / 확인 / 추가 문의] 답장을 써줘.
톤: 정중하고 짧게. 불필요한 인사말 없이.

[원본 메일 붙여넣기]
```

---

## 🗒️ 회의록 정리

```
아래 회의 내용을 정리해줘.

1. 결정된 사항
2. 액션 아이템 (담당자 · 기한 포함)
3. 다음 회의 안건

[회의 메모 또는 STT 텍스트 붙여넣기]
```

---

## 📊 엑셀 · 데이터 질문

> 파일을 Claude Code 탭에 드래그해서 넣고 아래처럼 질문하세요

```
이 파일에서 [원하는 정보]를 알려줘.
```

**예시 질문 모음**
- `금액이 가장 큰 항목 5개 알려줘`
- `이 VLOOKUP 수식이 뭘 하는 건지 설명해줘`
- `날짜 컬럼이 뒤죽박죽인데 오름차순으로 정렬해줘`
- `이 표에서 [부서명] 데이터만 뽑아줘`

---

## 📑 보고서 구조 잡기

```
[주제] 보고서 목차를 잡아줘.
- 보고 대상: [CFO / 팀장 / 전체 팀]
- 목표 분량: [A4 몇 장]
- 반드시 포함할 내용: [핵심 포인트 1~2개]
```

---

## 🔍 어려운 용어 · 규정 해석

```
아래 내용을 재무팀 실무자가 이해할 수 있게 쉽게 설명해줘.
전문 용어는 풀어서, 핵심 요점만 3줄로.

[조문 / 계약서 조항 / 용어 붙여넣기]
```

## ⭐ 잘 되는 프롬프트의 공식

| 요소 | 예시 |
|---|---|
| **역할** | "너는 재무팀 담당자야" |

| **형식** | "표로 정리해줘 / 5문장으로 써줘" |
