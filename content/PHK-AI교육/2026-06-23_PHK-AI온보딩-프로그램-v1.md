## Week 1 — "AI가 뭔지, 어떻게 말 걸지"

### 목표
- AI에 대한 막연한 두려움·거부감 해소
- 좋은 프롬프트 vs 나쁜 프롬프트 직접 비교 체험
- Claude.ai 계정 생성 및 첫 대화

### 강의 내용 (30분)

**① AI 현주소 (10분)**
- ChatGPT·Claude·Gemini — 뭐가 다른가
- Pizza Hut 외식업계 AI 도입 실제 사례 (Yum! Brands AI 주문·수요예측)
- "잘 쓰는 사람이 이기는 도구" — 평준화 vs 차별화

**② 프롬프트 기초 (20분)**
- 나쁜 프롬프트: `"메일 써줘"`
- 좋은 프롬프트: **역할 + 맥락 + 형식 + 분량**
  ```
  너는 피에이치코리아 재무팀 직원이야.
  거래처에 미지급금 확인 요청 메일을 써줘.
  - 수신: 거래처 담당자
  - 톤: 정중하고 간결하게
  - 분량: 5문장 이내
  ```
- 실습: 각자 자기 업무 1가지를 프롬프트로 표현해보기

### 실습 (15분)
- Claude.ai 가입 (무료 플랜 — claude.ai)
- 직접 작성한 프롬프트로 첫 대화
- 결과 팀 공유 (좋았던 것 / 이상했던 것)

### 과제
- 이번 주 업무 중 AI한테 물어볼 것 1가지 메모해오기

## Week 2 — "Finance 업무에 AI 쓰기"

### 목표
- Finance 실무에서 AI를 반복적으로 쓸 수 있는 패턴 체득
- 각자 실제 업무 1건을 AI로 처리하는 성공 경험

### 강의 내용 (30분)

**Finance 업무 유형별 AI 활용법**

| 업무 유형 | 활용 방법 | 예시 |
|---|---|---|
| 메일 작성 | 초안 생성 + 톤 조정 | 거래처 지급 안내, 팀 공지 |
| 회의록 정리 | STT 텍스트 → 요약·액션아이템 | 주간 미팅, 임원 보고 |
| 엑셀 수식 | 수식 설명·생성 | VLOOKUP 오류 해석, 조건부서식 |
| 보고서 구조 | 목차 제안 + 섹션 초안 | 월간 손익 보고, CFO 브리핑 |
| 법령·규정 해석 | 복잡한 조문 쉽게 설명 | 세금 규정, 계약서 조항 |

**주의사항**: 회사 기밀 수치·거래처명은 익명 처리 후 입력

### 실습 (15분)
- 과제로 가져온 업무 1건을 AI로 실제 처리
- 3명 자원받아 결과물 팀 공유

### 과제
- 이번 주 업무 중 AI 쓴 것 스크린샷 1장 (다음 주 공유)

## Week 3 — "VSCode + Claude Code 확장 설치"

> ✅ **터미널 직접 입력 없음** — VSCode 마켓플레이스에서 확장 설치 후 탭 열어서 채팅만 하면 됩니다.  
> 설치 자체가 어렵지 않으나, 7명 동시 설치 시 계정 연결 단계에서 막힐 수 있으니 충분한 지원 시간 확보.

### 사전 준비 (세션 전 배포)
- **사전 과제**: 없음 (세션 당일 함께 설치)
- **준비물**: 개인 노트북 (Windows), Anthropic 계정 (Week 1에 생성한 것)
- **IT 사전 협조**: VSCode 설치 권한 확인 (사내 PC 정책)

### 강의 내용 (20분)

**① Claude Code가 뭔가, Claude.ai와 뭐가 다른가 (10분)**

| | Claude.ai (웹) | Claude Code (확장) |
|---|---|---|
| 접근 | 브라우저 | VSCode 탭 |
| 특기 | 대화·글쓰기 | **파일 읽기·편집·생성** |
| 사용 방식 | 채팅창 | 채팅 + 파일 작업 |
| 예시 | "메일 초안 써줘" | "이 엑셀 데이터 정리해줘" |

핵심 차이: Claude Code는 **내 파일을 직접 보고 작업**할 수 있다.

**② 설치 순서 (10분)**
1. VSCode 설치: [code.visualstudio.com](https://code.visualstudio.com) → Download
2. 확장 마켓플레이스 열기: 좌측 사이드바 블록 아이콘 클릭
3. "Claude Code" 검색 → Anthropic 공식 확장 → Install
4. 탭 열기: 상단 메뉴 또는 `Ctrl+Shift+P` → "Claude Code: Open"
5. 로그인: Anthropic 계정으로 인증

### 실습 (30분)
- 7명 전원 설치 완료 확인
- 첫 대화 예시:
  > "안녕, 나는 피에이치코리아 재무팀이야. 오늘 해야 할 업무 메일 하나 써줄게."
- 파일 작업 체험: Week 2에서 쓴 엑셀 파일 하나 Claude Code에 끌어다 놓고 질문
  > "이 파일에서 금액이 가장 큰 항목 3개 알려줘"

### 강사 준비사항
- VSCode + Claude Code 확장 설치 가이드 1페이지 (스크린샷 포함) 출력 배포
- 계정 연결 오류 시 해결법 (브라우저 팝업 차단 해제 등) 미리 파악

## Week 4 — "첫 자동화 1개 완성"

### 목표
- 각자 반복 업무 1개를 Claude Code로 해결하는 경험
- 결과물 발표로 팀 내 사례 축적

### 사전 준비 (Week 3 종료 후 배포)
- **과제**: 내 업무 중 "매번 하는데 귀찮은 것" 1가지 선정해오기
- 예시: 엑셀 형식 통일, 메일 분류, 보고서 표 정리

### 세션 구성 (60분)

**① 자동화 설계 원칙 (15분)**
- Claude Code에게 일 시키는 법: **"뭘 원하는지"를 구체적으로**
- 좋은 지시: `"이 CSV 파일의 금액 컬럼을 원화 형식으로 바꾸고 새 파일로 저장해줘"`
- 반복 작업은 스크립트로 저장 → 다음에도 재사용

**② 각자 실습 (30분)**
- 선정한 업무 1개를 Claude Code로 처리
- 강사(Ethan)가 순회 지원

**③ 팀 발표 (15분)**
- 자원 3~4명: 5분 시연
  - 어떤 업무였나 / AI한테 어떻게 말했나 / 결과는

### 수료 기준
- 7명 전원 Claude Code 설치 완료 (Week 3 기준)
- 각자 자동화 1건 시도 (성공 여부 무관)
