# 세션 2: 아웃룩을 AI 두뇌와 연결하는 법
### "Ethan은 메일을 어떻게 쌓고, 쓰는가"

---

## 먼저, 우리 모두의 현실

하루에 메일 수십 통이 옵니다.

> 3개월 전에 거래처랑 어떤 조건으로 얘기했더라?  
> 저번에 부장님이 뭐라고 하셨지?  
> 이 건 내가 언제 회신했지?

아웃룩 검색해봅니다. 키워드가 기억 안 납니다.  
스크롤을 내립니다. 못 찾습니다.  
결국 **그 사람한테 다시 물어봅니다.** 😅

이걸 해결하는 게 이번 세션의 주제입니다.

---

## 비유로 이해하기 — "개인 비서 + 서류 창고"

일반적인 메일함 = **받은 편지함 하나에 모든 게 쌓이는 창고**
- 잘 정리된 것도 없고
- 찾기도 어렵고
- 읽고 나면 머릿속에서 사라짐

Ethan의 시스템 = **비서가 옆에 앉아서 메일을 함께 보는 구조**

```
① 메일이 오면 → 비서(AI)가 자동으로 따로 보관
② 중요한 것은 → 핵심만 요약해서 정리
③ 나중에 물어보면 → 즉시 꺼내줌
④ 메일 써야 할 때 → "Ethan 스타일대로" 초안을 써줌
```

---

## 실제로 어떻게 작동하나 (기술 설명 없이)

### STEP 1. 메일이 오면 → 자동으로 보관된다

Ethan의 아웃룩 메일은 매일 자동으로 **별도 저장소(vault)** 에 저장됩니다.

마치 파일 캐비닛처럼요.  
쌓이면 쌓일수록 나중에 더 잘 찾을 수 있습니다. **(현재 약 24,000통 보관 중)**

> "다 읽지 않아도 됩니다. 저장만 해두면 됩니다."

---

### STEP 2. 중요한 것은 → 요약 정리된다

받은 메일 중 업무적으로 중요한 것들은  
AI가 핵심 내용만 뽑아서 **노트 형태로 정리**해 둡니다.

예시:

> ❌ 원본 메일 (8단락짜리 긴 메일)
>
> ✅ 정리된 노트  
> `거래처: A업체 / 논의 내용: 단가 조정 요청 / 결론: 7월 협의 예정 / 담당: 담당자`

한 달 후에 "A업체 단가 건 어떻게 됐지?" 하면 → 즉시 꺼낼 수 있습니다.

---

### STEP 3. 나중에 물어보면 → 즉시 찾아준다

3개월 전 일도 괜찮습니다.  
아웃룩 검색처럼 키워드를 정확히 기억할 필요가 없습니다.

> "6월에 부장님이랑 얘기한 결제 조건 관련 내용 있어?"

AI가 저장된 메일과 노트에서 찾아줍니다.

이건 마치 **내가 나한테 잘 정리된 보고서를 써두는 것**과 같습니다.

---

### STEP 4. 메일 써야 할 때 → "Ethan 스타일로" 초안을 써준다

이게 가장 실용적인 부분입니다.

Ethan의 경우, AI가 과거 **발송 메일 334통을 분석**해 말투를 학습했습니다.  
그래서 맥락만 말해도 Ethan 스타일 그대로 초안이 나옵니다.

> 입력: "거래처 담당자한테 이번 달 정산 확인 부탁 메일 써줘. 정중하고 짧게."
>
> Claude Code 초안:
> ```
> 안녕하세요, 담당자님
>
> 이번 달 정산 내역 확인 부탁드립니다.
> ...
>
> 감사합니다.
> Ethan.
> ```

본인 스타일을 이미 알고 있으니  
"항상 수고 많으십니다" 같은 어색한 말은 절대 안 씁니다.

---

## 핵심 정리 — "AI가 뭘 해주는가"

| 상황 | 기존 방식 | AI 연동 방식 |
|---|---|---|
| 과거 메일 찾기 | 아웃룩 스크롤 + 기억에 의존 | 자연어로 물어보면 즉시 |
| 중요 내용 기억 | 내가 따로 메모하거나 까먹음 | AI가 자동 요약 보관 |
| 메일 쓰기 | 처음부터 직접 작성 | 초안 받아서 수정만 |
| 업무 인수인계 | 메일 다 뒤지기 | 정리된 노트 바로 공유 |

---

## "그게 Ethan이니까 가능한 거 아닌가요?"

아닙니다. 규모만 다를 뿐 원리는 동일합니다.

**지금 당장 시작할 수 있는 것**:

1. **메일 초안**: 보내야 할 메일 상황을 Claude Code에서 설명 → 초안 받기
2. **메일 요약**: 긴 메일을 붙여넣고 "핵심만 3줄로" 요청
3. **답장 초안**: 받은 메일을 붙여넣고 "이렇게 답장해줘" 요청

VSCode에 Claude Code 확장을 깔면 끝입니다.  
명령어도 없고, 코딩도 없습니다. **그냥 말로 시키면 됩니다.**

---

## 오늘 실습 — VSCode Claude Code 확장에서 해봅시다

**실습 A — 메일 초안**
> VSCode Claude Code 탭을 열고:  
> "수신자 직급 / 메일 목적 / 핵심 내용" 3가지를 입력  
> → 초안 받아서 수정만 하기

**실습 B — 긴 메일 요약**
> 최근 받은 긴 메일 1통을 Claude Code 탭에 붙여넣고:  
> "이 메일의 핵심 3가지와 내가 해야 할 액션을 정리해줘"

**실습 C — 답장 초안**
> 답장해야 하는 메일 1통을 Claude Code 탭에 붙여넣고:  
> "정중하고 간결하게 수락 답장 써줘"
